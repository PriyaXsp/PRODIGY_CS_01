# Caesar Cipher Encryption

A simple Python project that encrypts and decrypts text using the Caesar Cipher algorithm.

## Features
- Encrypt text
- Decrypt text
- User-defined shift value
- Simple console-based interface

## Technologies Used
- Python

## How to Run
1. Run `crypto.py`
2. Enter the message.
3. Enter the shift value.
4. Choose Encrypt or Decrypt.
